# 📊 Sales Data Pipeline (Python)

##  Project Overview

This project automates the consolidation of quarterly sales data from multiple Excel files and generates a structured summary report with a visualization chart.

The objective is to simulate a simple data processing pipeline that aggregates product sales across different months and produces a consolidated output file.

---

## Technologies Used

- Python 3
- openpyxl
- Excel (.xlsx)

---

## Features

- 📥 Load multiple Excel files (October, November, December)
- 🔄 Aggregate sales data by product
- 🧠 Store structured data using Python dictionaries
- 📊 Generate a consolidated Excel report
- 📈 Automatically create a bar chart visualization
- 💾 Export final quarterly report (`total_ventes_trimestre.xlsx`)
